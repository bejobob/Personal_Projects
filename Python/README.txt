SETUP

To set up EpicPaster, create a shorcut to the .exe file and place it in your startup folder. To do this, on your computer desktop screen, right-click, 
select New->Shortcut. Enter the file path to EpicPaster.exe. Press win+R on your keyboard to open the Run application, type "shell:startup", and hit enter.
Move the shortcut you just created to the startup folder that opened. Next time you start your computer, EpicPaster will run automatically.


USAGE

Once EpicPaster has been properly set up (see SETUP), the following keybinds will be active:

ctrl+shift+1: paste text in all caps. 
    ex: this is a test -> THIS IS A TEST
ctrl+shift+2: paste text in lowercase.
    ex: This is a TEST -> this is a test
ctrl+shift+3: paste text in sentence case.
    ex: this is a. test! TO SEE WHAT HAPPENS. -> This is a. Test! To see what happens.
ctrl+shift+4: paste text without formatting.
    ex: [in italics] this is a test -> this is a test

ctrl+1: change selected text to all uppercase
    ex: see example for ctrl+shift+1
ctrl+2: change selected text to all lowercase
    ex: see example for ctrl+shift+2
ctrl+3: change selected text to sentece case
    ex: see example for ctrl+shift+3
ctrl+4: keep casing of selected text, remove special formatting
    ex: see example for ctrl+shift+4

NOTE THAT ALL EPICPASTER KEYBINDS WILL REMOVE SPECIAL FORMATTING (ITALICS, BOLD, UNDERLINE, ETC).
